Steps

Numbered installation or wizard steps with done/pending states. A vertical connector line links steps and turns green as steps complete. Use dcr-steps__step--done on completed steps. Step content can include plain text, code snippets, or any other markup.

  1. Install the package

    Add the panel to your project via pip.
    pip install dj-redis-panel
  2. 2

    Add to INSTALLED_APPS

    Register the app in your Django settings.
    INSTALLED_APPS = [ # ... "dj_redis_panel", ]
  3. 3

    Restart & visit the admin

    Restart your Django server. The panel appears automatically in Control Room.

Spacing

Five-step scale used for padding, margin, and gap throughout the design system. Always reference a token - never hardcode pixel values.

--dcr-space-sm  ·  8px
--dcr-space-md  ·  16px
--dcr-space-lg  ·  24px
--dcr-space-xl  ·  48px
